Specifications

  1. install stuff
  2. clone this precourse
  3. copy the starter code into a new folder
  4. git init in that folder
  5. create new repo on your account
  6. set the remote in the new folder
  7. push the starter code
  8. change gh-pages to build from master
  9. put a link to the live demo in the repo description
  10. edit the code on your computer
    • using visual-studio
    • live-server
    • chromedev
  11. follow the instructions in the readme
    • make a simple thing in html (or we provide?)
    • make a render method that copies it
    • make a button that renders the thing
    • challenge: make a method that reads and replicates a dom element
  12. once you've done those things, fill in the blanks in the readme
  13. push it
  14. add the repo and live demo to your portfolio
